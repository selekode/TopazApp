Guía de instalación

Requisitos previos

  • Sistema operativo: Windows. Topaz solo funciona en Windows porque la base de datos se guarda en la ruta fija C:\Topaz\data.
  • Java: Debes tener instalada la versión Java 23 (JDK 23). Comprueba la instalación con:
java -version

Ejecución de Topaz

Para ejecutar Topaz, abre cmd, navega al directorio donde está el .jar y ejecuta:

cd C:\ruta\del\jar
java -jar topaz-25.9.1-SNAPSHOT.jar

Notas adicionales

  • Asegúrate de que el usuario de Windows que ejecute Topaz tenga permisos de lectura/escritura en C:\Topaz\data.
  • Si tienes problemas con la versión de Java, instala JDK 23 desde la web oficial de Oracle.
